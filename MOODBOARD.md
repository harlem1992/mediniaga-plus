# Mediniaga Plus — Website Mood Board & Design Direction

## 🎯 Design Philosophy: McKinsey × Medical Grade

> Clean, confident, data-driven. Think McKinsey & Company's website meets a premium medical brand.

---

## 🎨 Color Palette

```
Primary Dark:  #1a1a2e    (Navy — headers, footer, dark sections)
Brand Teal:    #1fbdd5    (Accent — CTAs, highlights, links)
Teal Dark:     #0d7377    (Hover states)
Light BG:      #f5f5fa    (Alternate section backgrounds)
Border:        #e8e8f0    (Cards, dividers)
Text Primary:  #16213e    (Body text)
Text Muted:    #8a8a9a    (Secondary text)
White:         #ffffff    (Main background)
```

## 🔤 Typography

| Element | Font | Weight | Size |
|---------|------|--------|------|
| **Headlines** | Lato | 900 (Black) | 3.2rem / 2.5rem / 1.8rem |
| **Subheads** | Lato | 700 (Bold) | 1.3rem / 1.1rem |
| **Body** | Assistant | 400 (Regular) | 1.05rem |
| **Labels** | Lato | 700 (Bold, uppercase) | 0.85rem |
| **Stats** | Lato | 900 (Black) | 2.5rem |

## 📐 Layout Grid

- **Max width:** 1200px
- **Gutters:** 24px
- **Grid:** 4-column (mobile) → 12-column (desktop)
- **Whitespace:** Generous. 80px section padding minimum.
- **Cards:** 280px minimum width, 12px radius, subtle shadow

## ✨ Key Design Elements

| Element | Style |
|---------|-------|
| **Hero** | Full-width with gradient background, big bold headline, CTA button |
| **Stats bar** | Dark navy section with animated counting numbers |
| **Product cards** | White cards with image, category label, title, price, CTA |
| **Industry cards** | Icon + title + description, colored left border accent |
| **Navigation** | Fixed top, transparent → scrolled white with shadow |
| **Buttons** | Pill-shaped (50px radius), teal primary, outlined secondary |
| **Footer** | Dark navy, 4-column grid, copyright bar at bottom |

## 🖼️ Imagery Style

Clean studio photography of products on white backgrounds. Icon-based industry representations. Charts and data visualizations for statistics.

## 📱 Responsive Behavior

- **Desktop:** Full multi-column layouts
- **Tablet:** 2-column grids
- **Mobile:** Single column stack with hamburger menu
